# Time Series Prediction of Sepsis

This project focuses on predicting sepsis using time-series patient data. It was developed as part of a machine learning lab in the Data Engineering program at KTH.

## Overview
Sepsis is a life-threatening condition that requires early detection. In this project, time-series data is analyzed to build a machine learning model capable of predicting the onset of sepsis.

## Objectives
- Analyze and preprocess time-series medical data
- Train machine learning models for prediction
- Evaluate model performance

## Features
- Data preprocessing and cleaning
- Handling time-series data
- Model training and evaluation
- Performance analysis

## Technologies
- Python
- NumPy
- Pandas
- Scikit-learn

## Results
The project demonstrates how machine learning techniques can be applied to time-series healthcare data for predictive analysis.

## About
This project was developed as part of coursework in Data Engineering at KTH Royal Institute of Technology. It reflects practical experience in machine learning, data analysis, and working with real-world datasets.
